# Deployment Contract

This project belongs exclusively to the `SeniorEquityAlliance` GitHub organization.

The only allowed GitHub repository for this website is `SeniorEquityAlliance/seanor`.

The local project folder is `~/Desktop/Codejects/SeniorEquityAlliance_website`.

This site is intended for GitHub Pages hosting from the `SeniorEquityAlliance/seanor` repository.

Never push, publish, deploy, copy, mirror, or configure this website under any other organization, repository, Pages site, or domain.

Before any push, verify that the git remote origin URL points to `SeniorEquityAlliance/seanor`.

Before any Pages-related change, verify that the target GitHub organization is `SeniorEquityAlliance`.

Do not reference or deploy this website to TavraOS, Hagerlabs, personal GitHub Pages, or any unrelated project.

Any future backend service must use Parse hosted on Back4App unless explicitly changed by the project owner.

If backend credentials are ever required, never hardcode secrets in frontend files.
